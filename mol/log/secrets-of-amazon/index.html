以下、気になったところ抜粋

2005年のアマゾンの売り上げは日本円で1兆円を超える。
売上高の55%は米国（日本は10%で1000億円くらいの売上高とされる）
全世界で登録ユーザーは4700万に及ぶとされる。
2005年の秋の時点でアマゾンは年間3億ドル（約345億円）をウェブサイトのデザインや開発に投資している。

アマゾンの成長に何よりも貢献した技術は
リコメンデーションとアソシエイト・プログラムの2つ。
リコメンデーションのアルゴリズムは商品間協調フィルタリング(Item-to-Item Collaborate Filtering)
ユーザー数4700万のマッチングをユーザーが許容できる0.5秒以内に行っている。
2005年、アソシエイト・プログラムに参加しているユーザーは全世界で80万人にものぼる。

Description
Amazon.com is looking for a talented designer with the ability to create clean, branded, web-ready designs within the context of existing style guide conventions. The ideal candidate will possess the ability to design graphics independently, as well as take direction in a team-based environment. This position requires strong organizational skills, web expertise, intrinsic design ability, refined typographic skills, utmost attention to detail, and acute customer focus.

Responsibilities:
* Designing strongly branded promotional graphics for a variety of clients.
* Creating graphical User Interface elements.
* Optimizing digital images for download efficiency.
* Developing clean and effective, graphic templates and styleguides.
* Responding to a variety of web and graphics related requests across departments.
* Fulfilling multiple project requests simultaneously while meeting tight deadlines.
* Manipulating pixel-level details to create elegant icons and imagery.
* Liaising with business teams to improve website elements.

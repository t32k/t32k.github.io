(X)HTML+CSSガイドライン3

CSSプロパティ記述順序っていろいろありますよね、

  • mozilla suggested order
  • スタイルシートを書く時のガイドライン
  • CSS2 Specificationでのプロパティの出現順序
↑こんな具合に。 どれを選べばいいんじゃーと悩んだので、いいとこ取りしてみた。 CSS Property Order Sheet(PDF)基本モジラベースで足りないところをkaminogoyaさんのガイドラインを足してみた。 『ページ出力』も足してみた。 『聴覚スタイルシート』は引いてみた。 CSS2.1で削除されたプロパティは引いてみた。
  • font-size-adjust
  • font-stretch
  • text-shadow
  • marker-offset
  • size
  • marks
  • page
んで思ったこと、これに同意↓ 「CSSの記述ルール」が不要になる時代はいつ来るのかなぁ……というぼやき話
